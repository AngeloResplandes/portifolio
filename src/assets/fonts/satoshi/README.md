# Satoshi Font Assets

Coloque aqui os arquivos da fonte Satoshi (.woff2 de preferência) para uso no projeto.

Recomendado:
- Satoshi-Variable.woff2 (peso 300–900)
- (Opcional) Satoshi-Italic.woff2

Após adicionar os arquivos, o CSS já está configurado para buscar `Satoshi-Variable.woff2` neste diretório.

Caso utilize nomes distintos, ajuste os caminhos em `src/css/index.css` nas regras `@font-face`.
